# Spartan-React demo

This demo shows the basic usage of Spartan-React

### How to use

Execute demo app with [Yarn](https://yarnpkg.com/lang/en/docs/cli/create/) to bootstrap the demo:

## Install node_modules

```bash
yarn install
```

## Run it locally

```bash
yarn dev
```
